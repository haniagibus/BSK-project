◆ generateKeys()

void org.example.KeyGenerator.generateKeys ( String PIN) throws Exception
inline

Generuje parę kluczy RSA i zapisuje je w odpowiednich lokalizacjach.

Metoda wykonuje następujące kroki:

  1. Wykrywa pendrive lub nośnik zewnętrzny
  2. Generuje parę kluczy RSA o rozmiarze 4096 bitów
  3. Szyfruje klucz prywatny przy użyciu PIN-u
  4. Zapisuje zaszyfrowany klucz prywatny na pendrive
  5. Zapisuje klucz publiczny lokalnie w formacie Base64
Parameters
PINPIN służący do szyfrowania klucza prywatnego
Exceptions
ExceptionW przypadku błędów kryptograficznych, I/O lub braku pendrive
