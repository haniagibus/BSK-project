◆ signPDF()

boolean org.example.PdfSigner.signPDF ( String filePath,
String pin )
inlinestatic

Podpisuje dokument PDF podpisem cyfrowym przy użyciu klucza RSA.

Metoda wykonuje następujące kroki:

  1. Wyszukuje zaszyfrowany klucz prywatny na nośnikach zewnętrznych
  2. Odszyfrowuje klucz prywatny przy użyciu PIN-u
  3. Generuje hash SHA-256 z pliku PDF
  4. Tworzy podpis cyfrowy RSA
  5. Zapisuje podpis do pliku z rozszerzeniem .sig
Parameters
filePathŚcieżka do pliku PDF do podpisania
pinPIN służący do odszyfrowania klucza prywatnego
Returns
true - jeśli operacja podpisania się powiodła, false - w przeciwnym razie
Exceptions
ExceptionW przypadku błędów kryptograficznych lub I/O
See also

◆ verifyPDFSignature()

boolean org.example.PdfSigner.verifyPDFSignature ( String filePath)
inlinestatic

Weryfikuje podpis cyfrowy dokumentu PDF na podstawie klucza publicznego.

Metoda wykonuje następujące kroki:

  1. Wczytuje klucz publiczny z pliku PEM
  2. Generuje hash SHA-256 z pliku PDF
  3. Wczytuje podpis z pliku .sig
  4. Weryfikuje podpis przy użyciu klucza publicznego
Parameters
filePathŚcieżka do pliku PDF do weryfikacji
Returns
true - jeśli podpis jest prawidłowy i dokument nie został zmodyfikowany, false - w przeciwnym razie
Exceptions
ExceptionW przypadku błędów kryptograficznych lub I/O
Note
Wymaga istnienia pliku podpisu o nazwie {filePath}.sig
